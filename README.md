Personal self-hosted dashboard built with an HDA (Hypermedia-Driven Application) stack. No frontend build step, no seperate API. Just type-safe Rust.

Stack:
* Axum: async web framework
* Maud: compile-time HTML templates via Rust macros
* HTMX: hypermedia controls for SPA-like interactions
* Hyperscript: lightweight inline scripting
* Bulma: CSS framework
* SQLx: async SQLite with compile-time query checking

Features:
* Zero-based paycheck budgeting: Allocate every dollar to a cause!
* ...and more to come!
